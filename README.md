# 🧠 AI-Powered Domain Intelligence Newsletter Agent

An agentic AI system that analyzes news and research papers to generate domain-specific newsletters with insights and learning recommendations.

## 🚧 Status

Work in progress

## 🎯 Goal

Build a multi-agent system using Google ADK for automated intelligence generation.

## 🛠️ Tech Stack (Planned)

* Google ADK
* Gemini API
* ChromaDB
* FastAPI
* Streamlit

## 📌 Features (Planned)

* News + research ingestion
* Semantic filtering
* Summarization
* Learning recommendations
* Newsletter generation
